HealMate Mobile Health Assistant

HealMate is a mobile healthcare assistant application developed as my Final Year Research Project. The application helps users identify possible health conditions based on symptoms and provides personalized healthcare recommendations.

One of the main features of the system is the Interactive Body Map, which allows users to select the affected body area and identify symptoms easily. The application then analyzes the selected symptom together with age and gender information to provide more personalized health guidance.

The system includes symptom severity assessment, confidence-based analysis, healthcare recommendations, product suggestions, health news updates, and saved result management. An admin dashboard was also developed to manage symptoms, severity questions, products, and healthcare content through Firebase.

This project was developed using Flutter and Firebase, focusing on improving healthcare accessibility through an easy-to-use mobile platform while strengthening my mobile application development, Firebase integration, and UI/UX design skills.

IoT-Based Smart Shelf

The IoT-Based Smart Shelf is an innovative system developed to help businesses track stock levels and item movements in real-time. It combines hardware and software to monitor inventory automatically, reducing errors and improving efficiency.

The system uses Arduino, ESP32, load cells, RFID tags, and LCD displays to detect product weight and identify items. I built a web dashboard using HTML, CSS, and JavaScript, connected to Firebase for real-time data visualization.

A unique point about this project is how it integrates multiple devices into one ecosystem, not only displaying stock levels but also identifying fast-moving and slow-moving items.

LAMP Studio - Furniture Visualization Application

LAMP Studio is a web-based furniture visualization application developed to improve the interior design experience by allowing designers and customers to create, customize, and visualize virtual room environments.

The system enables designers to create room layouts by entering room dimensions, selecting room shapes, applying colour schemes, and arranging furniture items using an interactive 2D drag-and-drop editor. Users can resize, customize colours, adjust opacity, and manage furniture placement according to their design requirements.

The application provides an advanced 3D visualization feature using Three.js, allowing users to convert 2D layouts into realistic interactive 3D scenes. Users can explore room designs, switch views, modify furniture colours, and preview how furniture will appear in real-world environments.

A secure authentication system with role-based access control was implemented for Designers and Admins. Designers can create, save, edit, and manage their designs, while administrators can monitor users and manage system data through the admin dashboard.

This project was developed using Next.js, React.js, Three.js, Tailwind CSS, Express.js, and MongoDB. The project enhanced my knowledge of full-stack development, 3D web technologies, UI/UX principles, database management, authentication, and interactive application development.

Vehicle Service & Repair Management System

The Vehicle Service & Repair Management System is a desktop application developed to improve the efficiency of managing vehicle service, repair, and maintenance records. The system replaces traditional manual record-keeping methods with a centralized digital solution.

The application allows employees to enter and manage vehicle servicing details, repair records, accident information, and customer-related data through an easy-to-use interface. This helps reduce paperwork, improve data accuracy, and speed up information retrieval.

An administrator module was developed to provide complete control over the system, including viewing, updating, and deleting records. Secure user authentication and database integration were implemented to ensure reliable and secure data management.

This project was developed using C#, Windows Forms, and SQL Server. It strengthened my knowledge of desktop application development, database management, CRUD operations, user authentication, and software design principles.
